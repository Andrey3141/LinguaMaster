# LinguaMaster v1.13.1

Современное приложение-тренажер для изучения иностранных слов с графическим интерфейсом на Python/Tkinter.

## 📋 О проекте

LinguaMaster - это полнофункциональное приложение для изучения иностранных слов с четырьмя режимами обучения, поддержкой 11 языков, гибридной озвучкой и системой умных подсказок.

### ✨ Ключевые возможности

- **4 режима обучения**: ручной перевод, тест, соотношение слов, перевод картинок
- **11 языков**: английский, русский, испанский, французский, немецкий, китайский, японский, корейский, португальский, итальянский, арабский
- **Гибридная озвучка**: Edge-TTS (нейросетевые голоса) + RHvoice (офлайн)
- **Умные подсказки**: автоматическое появление правильного ответа после N ошибок
- **Визуальное обучение**: привязка картинок к словам
- **Система категорий**: организация слов по темам
- **Статистика**: история обучения по дням с отслеживанием времени
- **Автоматическое сохранение**: все настройки сохраняются между сессиями

## 📖 Использование

### Основные режимы обучения

| Режим                | Описание                            |
|----------------------|-------------------------------------|
| **Ручной перевод**   | Введите перевод слова в поле ввода  |
| **Тест**             | Выберите правильный вариант из трех |
| **Соотношение**      | Сопоставьте слова из двух колонок   |
| **Перевод картинок** | Угадайте слово по изображению       |

### Управление

- **➕ Добавить слово** - добавить новое слово в словарь
- **📖 Показать словарь** - просмотр всех слов с фильтрацией
- **🎓 Метод обучения** - переключение между режимами
- **⚙️ Настройки** - настройка приложения
- **📁 Категория** - выбор категории для тренировки

### Настройки

#### Основные
- Автопереход к следующему слову
- **Умные подсказки** (с настройкой количества ошибок)
- Порог сложных слов

#### Озвучка
- Включение/отключение озвучки
- Автоматическая озвучка новых слов
- Громкость и скорость речи
- Выбор голосов для каждого языка

### Система подсказок

При включенных подсказках:
- После N ошибок правильный ответ появляется серым цветом
- Подсказка исчезает при начале ввода
- Подсказка восстанавливается при пустом поле
- Индикация статуса: "Подсказка через X ошибок", "Подсказка готова", "Подсказка активирована"

### Работа с картинками

1. При добавлении слова нажмите "📁 Выбрать картинку"
2. Выберите изображение в проводнике
3. Картинка скопируется в папку `data/pictures`
4. В режиме "Перевод картинок" слова с картинками будут показываться автоматически

## 📊 Статистика

Приложение отслеживает:
- Общее количество слов
- Изученные слова (сложность > 80%)
- Сложные слова (сложность < 50%)
- Прогресс за сегодня
- Точность ответов
- Время, проведенное в приложении
- История по дням

## 🔧 Решение проблем

### Нет звука при озвучке
1. Проверьте, включена ли озвучка в настройках
2. Убедитесь, что установлен RHVoice (для офлайн-озвучки)
3. Проверьте интернет-соединение (для Edge-TTS)

### Не сохраняются настройки
- Проверьте права на запись в папку `data/`
- Убедитесь, что папка существует

### Не отображаются картинки
- Проверьте, что картинки есть в папке `data/pictures/`
- Поддерживаемые форматы: PNG, JPG, JPEG, GIF, BMP
