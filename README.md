## 📖 О проекте

**LinguaMaster** — это полнофункциональное приложение для изучения иностранных слов, созданное с фокусом на удобство и эффективность.

### ✨ Возможности

| Фича | Описание |
|------|----------|
| 🎯 **4 режима обучения** | Ручной перевод, тест, соотношение слов, перевод картинок |
| 🌐 **11 языков** | Английский, русский, испанский, французский, немецкий, китайский, японский, корейский, португальский, итальянский, арабский |
| 🔊 **Гибридная озвучка** | Edge-TTS (нейросетевые голоса онлайн) + RHVoice (офлайн) |
| 💡 **Умные подсказки** | Автоматическое появление правильного ответа после N ошибок |
| 🖼️ **Визуальное обучение** | Привязка картинок к словам для лучшего запоминания |
| 📊 **Статистика** | История обучения, точность ответов, время в приложении |
| 💾 **Автосохранение** | Все настройки и прогресс сохраняются между сессиями |

## 📥 Установка и запуск

### ⚡ Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Andrey3141/LinguaMaster.git
cd LinguaMaster

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите приложение
python main.py
```

### 📦 Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Интернет (для Edge-TTS озвучки)

### 🔊 Настройка озвучки (опционально)

<details>
<summary>🔽 RHVoice (офлайн-озвучка)</summary>

```bash
# Ubuntu/Debian
sudo apt-get install rhvoice

# Arch Linux
sudo pacman -S rhvoice

# Другие ОС: https://github.com/RHVoice/RHVoice
```
</details>

<details>
<summary>🔽 Edge-TTS (онлайн-озвучка)</summary>

Устанавливается автоматически через `requirements.txt`. Требуется интернет-соединение.
</details>

## 🎮 Использование

### Основные режимы

| Режим | Для чего подходит |
|-------|------------------|
| 🔤 **Ручной перевод** | Для отработки написания и запоминания |
| ❓ **Тест** | Для быстрой проверки знаний |
| 🔗 **Соотношение** | Для развития ассоциативного мышления |
| 🖼️ **Перевод картинок** | Для визуалов и запоминания через образы |

### 🔑 Горячие клавиши и управление

- **➕ Добавить слово** — пополнение словаря
- **📖 Показать словарь** — просмотр и фильтрация слов
- **🎓 Метод обучения** — переключение режимов
- **⚙️ Настройки** — кастомизация под себя

> 💡 **Совет:** Включите «Умные подсказки» в настройках, чтобы приложение помогало вам при затруднениях.

### 🖼️ Работа с картинками

1. При добавлении слова нажмите **«📁 Выбрать картинку»**
2. Выберите изображение (PNG, JPG, GIF, BMP)
3. Картинка автоматически скопируется в `data/pictures/`
4. В режиме «Перевод картинок» слово будет показано с изображением

## 🔧 Решение проблем

<details>
<summary>❌ Нет звука при озвучке</summary>

1. Проверьте, включена ли озвучка в настройках (`⚙️ Настройки → Озвучка`)
2. Для офлайн-озвучки: убедитесь, что установлен RHVoice
3. Для онлайн-озвучки: проверьте интернет-соединение
4. Попробуйте сменить голос в настройках
</details>

<details>
<summary>❌ Не сохраняются настройки</summary>

- Проверьте права на запись в папку `data/`
- Убедитесь, что папка `data/` существует
</details>

<details>
<summary>❌ Не отображаются картинки</summary>

- Проверьте, что файлы есть в `data/pictures/`
- Поддерживаемые форматы: PNG, JPG, JPEG, GIF, BMP
- Имена файлов не должны содержать спецсимволов
</details>
