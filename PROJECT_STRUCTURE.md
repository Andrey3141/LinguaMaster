# VOCABULARY TRAINER - Архитектура проекта
## LinguaMaster v1.13.1 (2026)

Современное приложение-тренажер для изучения иностранных слов с графическим интерфейсом на Python/Tkinter.  
Полная поддержка озвучки слов через Microsoft Edge-TTS и RHvoice.  
**Четыре режима обучения:** ручной перевод, тест, соотношение слов и перевод картинок.  
**Умные подсказки:** автоматическое появление правильного ответа после заданного числа ошибок.  
**Визуальное обучение:** возможность прикреплять картинки к словам для ассоциативного запоминания.

## НОВОЕ В v1.13.0:
1. **РЕЖИМ ПЕРЕВОДА КАРТИНОК**
   - Визуальное обучение через ассоциативные картинки
   - Поддержка двунаправленного перевода (изучаемый ↔ родной)
   - Автоматический выбор слов с прикрепленными картинками
   - Черный фон для лучшего восприятия изображений
   - Сообщение об отсутствии картинок в категории

2. **СИСТЕМА УПРАВЛЕНИЯ КАРТИНКАМИ**
   - Добавление картинок при создании слова
   - Предпросмотр выбранного изображения
   - Автоматическое копирование в папку data/pictures
   - Переименование файла по названию слова
   - Защита от дублирования (добавление счетчика)

3. **УМНЫЙ ВЫБОР СЛОВ ДЛЯ КАРТИНОК**
   - Фильтрация только слов с прикрепленными изображениями
   - Исключение повторений (не показывает одно и то же слово дважды)
   - Работа с категориями
   - Интуитивное сообщение при отсутствии картинок

4. **УЛУЧШЕНИЯ ИНТЕРФЕЙСА**
   - Увеличенный диалог добавления слова (600x700) с прокруткой
   - Предпросмотр картинки перед добавлением
   - Копирование файла только при подтверждении добавления
   - Информативное сообщение о папке назначения

5. **ДВУНАПРАВЛЕННОСТЬ**
   - Одна картинка работает для обоих языков
   - При смене языка ожидается перевод на соответствующий язык
   - Корректная работа подсказок для обоих направлений

## ОСОБЕННОСТИ АРХИТЕКТУРЫ:
- **MVC-подход**: Model - View - Controller
- **Четыре режима обучения**: динамическое переключение
- **Гибридная озвучка**: Edge-TTS + RHvoice
- **Умные подсказки**: настраиваемый порог ошибок
- **Визуальное обучение**: интеграция с изображениями
- **Единое хранилище настроек**: app_settings.json
- **Персистентность картинок**: отдельная папка pictures
- **История статистики**: сохранение всех дней + время
- **Асинхронность**: отдельные потоки для TTS
