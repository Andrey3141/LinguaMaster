## [v1.9.0] - 2026-02-23

### Added
- Новая система статистики с историей всех дней обучения:
  - Сохраняется статистика за каждый день, а не только текущий
  - Автоматическая конвертация старых файлов stats.json в новый формат
  - Возможность просмотра прогресса по дням
- Отслеживание времени в приложении:
  - Добавлено поле time_spent для каждого дня
  - Время измеряется в секундах и автоматически суммируется
  - При выходе из приложения время сохраняется
  - При повторном запуске время продолжает накапливаться
- Новая структура stats.json:
  - history: массив со статистикой за все дни
  - current: текущий день для быстрого доступа
- Новые методы в модели:
  - `get_stats_history()` - возвращает всю историю статистики
  - `get_total_stats()` - возвращает суммарную статистику за все время (общее количество слов, правильных ответов, точность в %, общее время в приложении, количество дней обучения)
- Технические улучшения:
  - Сохранение времени при каждом вызове save_stats()
  - Обновление времени при закрытии приложения
  - Автоматический сброс времени сессии после сохранения
  - Форматирование времени для отображения (часы, минуты, секунды)

### Fixed
- Больше нет перезаписи статистики при смене дня
- Корректная работа с разными временными зонами
- Сохранение данных даже при аварийном завершении
