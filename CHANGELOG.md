## [v1.10.0] - 2026-02-23
### Новый режим: Соотношение слов

#### Режим сопоставления
1. Интерфейс с двумя колонками
   - Слева: 3 слова на изучаемом языке
   - Справа: 3 слова на родном языке (в случайном порядке)
   - Яркие заголовки для каждой колонки
   - Визуальная индикация выбранных кнопок

2. Механика игры
   - Игрок нажимает на слово слева, затем на слово справа
   - При правильном сопоставлении пара блокируется и подсвечивается зеленым
   - При ошибке пара подсвечивается красным на секунду
   - Прогресс отображается внизу: Сопоставлено: X/3

3. Умная генерация
   - Автоматический выбор 3 случайных слов из текущей категории
   - Проверка на минимальное количество слов (нужно минимум 3)
   - Сообщение, если недостаточно слов для режима

#### Кнопка проверки
1. Функция Проверить все
   - Подсвечивает правильные пары желтым цветом
   - Помогает игроку, если он затрудняется
   - Не дает готовых ответов, только подсказки

2. Автоматическая проверка
   - При сопоставлении всех трех пар показывается анимация успеха
   - Автоматический переход к следующему заданию (если включено)

#### Технические улучшения
1. Новый файл match_panel.py
   - Полностью независимая панель для режима соотношения
   - Логика выбора и сопоставления
   - Визуальная обратная связь

2. Интеграция с существующей системой
   - Добавлен метод generate_match_words
   - Обновлен метод switch_training_panel
   - Обновлен метод next_word для поддержки нового режима
   - Обновлен диалог выбора метода обучения

#### Интерфейсные улучшения
1. Три метода обучения в одном диалоге
   - Ручной перевод
   - Тест (выбор варианта)
   - Соотношение (сопоставление)

2. Визуальная обратная связь
   - Зеленый: правильное сопоставление
   - Красный: ошибка
   - Желтый: подсказка (правильная пара)

## [v1.9.0] - 2026-02-23
### Статистика с историей и отслеживание времени

#### Новая система статистики
1. История всех дней обучения
   - Теперь сохраняется статистика за каждый день, а не только текущий
   - Автоматическая конвертация старых файлов stats.json в новый формат
   - Можно просматривать прогресс по дням

2. Отслеживание времени в приложении
   - Добавлено поле time_spent для каждого дня
   - Время измеряется в секундах и автоматически суммируется
   - При выходе из приложения время сохраняется
   - При повторном запуске время продолжает накапливаться

3. Новая структура stats.json
   - history: массив со статистикой за все дни
   - current: текущий день для быстрого доступа
   - Каждая запись содержит дату, слова, правильные ответы и время

#### Новые методы в модели
1. get_stats_history() - возвращает всю историю статистики
2. get_total_stats() - возвращает суммарную статистику за все время
   - Общее количество слов
   - Общее количество правильных ответов
   - Общая точность в процентах
   - Общее время в приложении
   - Количество дней обучения

#### Технические улучшения
1. Сохранение времени при каждом вызове save_stats()
2. Обновление времени при закрытии приложения
3. Автоматический сброс времени сессии после сохранения
4. Форматирование времени для отображения (часы, минуты, секунды)

#### Исправлено
1. Больше нет перезаписи статистики при смене дня
2. Корректная работа с разными временными зонами
3. Сохранение данных даже при аварийном завершении
